# AI Trading Assistant

A beginner project focused on using AI for trading research, market observation, and structured decision-making.

## About this project

I am building this project as part of my learning journey in AI, data analysis, and automation.

The goal of this project is to explore how AI can help traders:
- summarize market news
- organize watchlists
- track trading setups
- improve discipline
- avoid emotional decisions

## Problem

Many beginners take trades based on emotions or random tips.

## Solution

This AI Trading Assistant helps with:
- News summaries
- Trade checklists
- Learning support
- Trade journaling
